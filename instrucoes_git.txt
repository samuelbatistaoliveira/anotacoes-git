Curso Git e GIthub

Comandos

git init
inicializa um repositório Git em uma pasta que ainda não tinha controle de versão.

git status
Mostra o estado atual do repositório, indicando arquivos modificados, adicionados ou não rastreados.

.gitignore
Para ignorar uma pasta, coloque o nome dela no arquivo .gitignore:

nome_do_diretorio/


Para ignorar todos os arquivos Python:
Ignora todos os arquivos com extensão .py

git config user.name 'Samuel Batista' 
Define o nome do autor dos commits.

git config user.email 'samuelbbatista32@gmail.com'
Define o e-mail do autor dos commits.


Configuração global (vale para todos os repositórios):

git config --global user.name "Samuel Batista"
git config --global user.email "samuelbbatista32@gmail.com"

git add
Adiciona arquivos à staging area (área de preparação), marcando quais alterações serão incluídas no próximo commit.

git add .
Adiciona todas as alterações da pasta atual e de suas subpastas à staging area.

git commit -m "commit um"
Cria um commit com as alterações que foram adicionadas à staging area através do comando git add.

git log
Exibe o histórico de commits do repositório, mostrando informações como autor, data, hash e mensagem de cada commit.

git config core.pager cat
Desabilita o pager e mostra tudo diretamente no terminal.

git config core.pager less
Habilita o pager "less", permitindo navegar pela saída de comandos longos.

git log -1
Exibe apenas o commit mais recente do repositório.

git log --oneline
Exibe o histórico de commits de forma resumida

git log --oneline -2
Exibe os 2 commits mais recentes de forma resumida

git log --before="2020-06-14"
Exibe os commits realizados antes da data informada.

git log --after="2020-06-13"
Exibe os commits realizados após a data informada.

git log --after="1 week ago"
Exibe os commits realizados na última semana.

git log --since="2 days ago"
Exibe os commits realizados nos últimos 2 dias.

git log --before="1 month ago" 
Exibe os commits realizados antes de um mês atrás.

git log --author="Samuel Batista"
Exibe apenas os commits realizados pelo autor informado.

git log --oneline --author="Samuel Batista"
Exibe de forma resumida apenas os commits realizados pelo autor informado.

git help log
